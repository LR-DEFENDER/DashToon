# Comic Strip Generator

This is a simple Comic Strip Generator project that allows users to input prompts and generate a comic strip based on those inputs.

## Table of Contents

- [Features](#features)
- [Usage](#usage)

## Features

- Input prompts for generating a comic strip.
- Dynamic generation of comic strip images based on user inputs.
- Bootstrap styling for a visually appealing interface.
- Responsive design allowing seamless experience on all size devices.
- Beautifully placed loading icons and GIFs for better user understandibility.
- Mandatory user input to generate images,shows a warning if all input fields are not populated by the user.

## Usage

- Enter all 10 prompts in the provided input fields.
- Click the "Generate Comic" button to create a comic strip.
- View the generated comic strip images below.
-Shows alert message if input is not prompted.
